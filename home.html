1. Wiki.js介绍

Wiki.js 是一个开源的、现代化的 Wiki 软件，旨在帮助用户创建功能强大、易于维护的团队知识库和文档。

Wiki.js 内置了多种的编辑器供你选择，如 Markdown、Visual Editor（即 WYSIWYG，所见即所得）、AsciiDoc、原始的 HTML 编辑器等。

 

以下是 Wiki.js 的一些特点和功能：

 

1. 现代化的界面：Wiki.js 提供了一个直观且现代化的界面，易于使用，用户可以轻松导航和查找信息。

 

2. 多语言支持：Wiki.js 支持多种语言，也支持同时使用多种语言。

 

3. 用户权限管理：管理员可以控制用户对内容的访问和编辑权限，确保只有授权人员可以对特定页面进行更改。默认情况下有两个组，即管理员组和访客组，当然我们也可以根据需要自定义其他用户组，之后在用户列表中找到对应用户并归入相应的组别中统一管理。

 

4. 多种存储：需要搭配 RDBMS 数据库使用，可以在后台设置进行更改，支持Amazon S3 对象存储或是其他兼容 S3 协议的存储服务、SFTP、Azure Blob Storage 等，也可以选择本地文件的存储方式，这样可以开启每日备份选项。

 

5.同步方式: 支持 Git 协议的存储与同步，可以设置双向同步，单向推送或单向拉取。

除了直接在注册好的 Wiki.js 站点上注册之外，Wiki.js 还集成了国外多个平台的身份验证，如 GitHub、Google、Slack 等，如果你想接入自己的身份认证服务，那么 Wiki.js 也预留了 OAuth2、CAS、Generic OpenID Connect 等多种认证服务接口。

 

6. 插件和扩展：Wiki.js 支持插件和扩展，允许用户根据需要添加额外的功能。

 

7. 全文搜索：内置全文搜索功能，帮助用户快速找到所需的信息。

 

8. 集成和 API 支持：Wiki.js 可以轻松地与其他工具和服务集成，例如 Git、Slack 等。

 

9. 主题定制：用户可以根据自己的需求和品牌风格定制 Wiki.js 的外观和感觉。
